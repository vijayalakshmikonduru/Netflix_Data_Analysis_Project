📊 Netflix Data Analysis Project

📌 Overview

This project focuses on analyzing Netflix content data using MySQL and visualizing insights through a Power BI dashboard.
The goal is to transform raw data into meaningful insights that help understand content distribution, trends, and business patterns.

🎯 Key Business Problems Solved

📺 What is the total number of shows available on Netflix?
🎬 How is content distributed between Movies and TV Shows?
⏱ What is the average duration of Movies vs TV Shows?
🌍 Which countries produce the most Netflix content?
🎭 Which genres dominate the platform?
📈 How has content production and addition changed over time?
🆕 Is Netflix focusing more on new or old content?
🔁 Are there duplicate titles in the dataset?

🛠 Tools & Technologies

MySQL – Data cleaning, transformation, and analysis
Power BI – Dashboard creation and data visualization
Dataset – Netflix titles dataset (CSV)

🗂 Project Structure

nf_db.sql → Database creation, data cleaning, and analysis queries
Views:
Total shows
Movies count
TV Shows count
Average duration
Genre trends
Country-wise production
Content trends over time
Power BI Dashboard (.pbix) → Visual representation of insights

🔄 Data Processing Steps

Created database and table (nf_data)
Imported CSV data into MySQL
Cleaned and transformed data:
Handled null/empty values
Converted date formats
Standardized year fields
Converted duration into numeric format
Created multiple views for KPIs and analysis

📊 Dashboard Highlights

Total number of shows
Movies vs TV Shows distribution
Average duration comparison
Top genres and countries
Year-wise and quarterly content trends
Content age analysis (old vs new)

This project demonstrates how raw data can be transformed into actionable insights using SQL and visualized effectively
